JavaScript 事件

JavaScript与HTML之间的交互式通过事件实现的.事件,就是文档或浏览器窗口中发生的一些特定的交互瞬间.

什么叫事件?

比如说,当我们点击一个按钮的时候.会弹出一个对话框.在JavaScript中,”点击”这个事情就看成一个事件.”弹出对话框”其实就是我们在点击事件中做的事.

  • 事件分为3部分:

    1. 这个事件是干什么的,是点击还是移动;
    2. 事件过程中都发生了什么;
    3. 目睹整个事件的用户;

应该能这样理解这就是一个”事件”.

  • 事件包括5大部分:

    1. 鼠标事件;
    2. 键盘事件;
    3. 表单事件;
    4. 编辑事件;
    5. 页面相关事件.

1.在script标签中调用;
2.在元素中调用;

鼠标事件

常用的鼠标事件如下图:

事件处理器的命名原则是:在事件名称前加上前缀on.

  1. 鼠标单击事件:onclick
    onclick是最常用的事件,如点击按钮弹出一个提示框.

  2. 鼠标移入和移出事件分别是:onmouseover和onmouseout
    onmouseover和onmouseout这两个是事件是形影不离的,都是共同使用来分别控制鼠标”移入“和”移出“.如下拉菜单导航中鼠标移入导航时会显示对应的二级下拉菜单,鼠标移出主导航时二级下拉菜单就会收起来.

  3. 鼠标按下和松开事件分别是onmousedown和onmouseup.onmousedown表示鼠标按下的一瞬间所触发的事件,而onmouseup表示鼠标松开的一瞬间所触发的事件.

键盘事件

常用键盘事件有3种:

  1. onkeypress事件是在键盘上某个键被按下到松开整个过程中触发的事件.

  2. onkeydown事件也是在键盘按下时触发.

  3. onkeyup事件是在键盘的某个键被按下之后松开的一瞬间触发的事件.

表单事件

常用的表单事件有4种:

  1. onfocus事件:表示获取焦点触发事件;

  2. onblur事件:表示失去焦点触发的事件;具有获得焦点和失去焦点事件的元素有:单行
    文本框text,多行文本框textarea,下拉列表select.

  3. onchange事件:当用户在单行文本框text和多行文本框textarea输入文本时,由于文本框内字符串的改变将触发onchange事件.

  4. onselect事件:当用户选中单行文本框text或多行文本框textarea的文本时，会触发onselect事件.onselect事件的具体过程是从鼠标按键被按下,到鼠标开始移动并选中内容的过程.这个过程并不包括鼠标键的放开.

编辑事件

常见的剪辑事件有3种:

  1. 赋值事件oncopy:在网页中复制内容时会触发复制事件oncopy,可以通过oncopy事件来禁止用户复制网页内容.

  2. 剪切事件oncut:在网页文本框等中选中的内容被剪切时会触发剪切事件oncut.

  3. 粘贴事件onpaste:在文本框中粘贴内容是会触发粘贴事件.

JavaScript Window

ECMAScript是JavaScript的核心,但如果要在web中使用JavaScript,那么浏览器对象模型(BOM)则无疑才是真正的核心.BOM 提供了很多对象,用于访问浏览器的功能,这些功能与任何网页内容无关.简单来说就是浏览器对象模型(BOM)使 JavaScript 有能力与浏览器”对话”.

Window 对象

BOM 的核心对象是 window,它表示浏览器的一个实例.在浏览器中 window 对象有双重角色,既是通过JavaScript 访问浏览器窗口的一个接口,又是 ECMAScript 规定的 Global 对象.意味着在网页中定义的任何一个对象、变量和函数,都以 window 作为其 Global 对象,因此有权访问 parseInt() 等方法.
全局变量是 window 对象的属性.
全局函数是 window 对象的方法.

昨天了解了JavaScript的简介,今天该继续了解JavaScript的数据结构和数据类型.

数据结构

标识符

指变量、函数、属性的名字,或者函数的参数.

  • 第一个字符必须是一个字母、下划线(_)或一个美元符号($);
  • 变量名不能包含空格、加号、减号等符号;
  • 标识符不能喝JavaScript中用于其他目的的关键字同名.

关键字

指在JavaScript语言中有特定含义,成为JavaScript语法中一部分的字.

常量

指一旦创建,值不可改变的特殊变量.主要用于为程序提供固定和精确的值,包括数字和字符串,如数字、逻辑值真（true）、逻辑值假（false）等都是常量。

变量

是存储信息的容器.

  1. 变量的命名:实际上是一个标识符,因此命名一个变量时也要遵循标识符的命名规则.
  • 第一个字符必须是字母、下划线（_）或美元符号,其后的字符可以是字母、数字或下划线、美元符号;
    • 变量名不能包含空格、加号、减号等符号；
    • 标识符不能和JavaScript中用于其他目的的关键字同名；

  1. 变量的声明与赋值
    所有的JavaScript变量都由关键字var声明.一个var也可以同时声明多个变量名,变量名之间必须用英文逗号’,’隔开.

  2. 变量的作用域:指某变量在程序中的有效范围,也就是程序中定义这个变量的区域.变量区域可分为:全局变量和局部变量.

JavaScript数据类型

定义:变量中存储的数据的类型.是弱数据的方式.

JavaScript数据类型有两大类,一是基本数据类型,二是特殊数据类型.

基本数据类型:
number:数字型
string:字符串型
boolean:布尔型

特殊数据类型:
null:空值型
undefined:未定义
转义字符

Number 类型

number是最基本的数据类型.不区别整型数值(int)和浮点型数据(float).

  • 整型数据:是整数没有小数的,是数据形式十进制整数来的,整数可以为正数、0或负数.
  • 浮点型数据:带有小数点的数据,还可以使用指数法来表示,即实数后跟随字母e或E,后面加上正负号.

String 类型

用于表示由零个或多个字符组成的序列.字符串可以由双引号(“)或单引号(‘)表示.

Boolean 类型

仅有两个值得特殊数据类型:true、false.这两个值与数字值不是一回事,因此true不一定等于1,而false也不一定等于0.

null 类型

是只有一个值的数据类型.null值表示一个变量被赋予了一个空值.
null不等同于空的字符串(“”)或0,因为空的字符串(“”)或0是存在的,但是null表示其不存在的.

undefined 类型

undefined表示一个变量仅声明过,但从未被赋值.

转义字符

专门表示非打印字符以及特殊符号.

\n    回车换行;
\t    Tab符号;
\f    换页;
\'    单引号;
\"    双引号;
\v    跳格（Tab，水平）;
\r    换行;
\\    反斜杠;
\OOO    八进制整数，范围为000~777;
\xHH    十六进制整数，范围为00~FF;
\uhhhh    十六进制编码的Unicode字符.

JavaScript简介

JavaScript,就是我们通常所说的JS.是一种嵌入到HTML页面中的脚本语言,由浏览器一边解释一边执行,是世界上最流行的编程语言.
这门语言可用于 HTML和 web,更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备.

JavaScript的特点

HTML页面是静态的,而JavaScript可以弥补HTML语言的缺陷,实现Web页面客户端的动态效果.JavaScript的作用有以下几点：

1、动态改变页面内容
HTML页面是静态的,一旦编写内容是无法改变的.JavaScript可以弥补这个不足,可以将内容动态地显示在网页中.

2、动态改变网页的外观
JavaScript通过修改网页元素的CSS样式,达到动态地改变网页的外观.

3、验证表单数据
我们常见的在各大网站中的注册中的验证功能,就是JavaScript实现的.

4、响应事件
JavaScript是基于事件的语言.例如点击一个按钮弹出一个对话框,就是鼠标点击触发的事件.

  • 页头引入(head标签内);
  • 页中引入(body标签内);
  • 元素事件中引入(标签属性中引入);
  • 引入外部JS文件;

指的就是在< head>< /head>标签内编写JavaScript.

表示在之间的是文本类型(text)javascript是为了告诉浏览器里面的文本是属于JavaScript语言.

页中引入JS

指的在< body>< /body>标签内编写JavaScript.

元素事件中引入JS

指的是在元素的某一个属性中直接编写JavaScript程序或调用JavaScript函数,这个属性指的是元素的”事件属性”.

引入外部JS文件

就是把JavaScript程序存放在一个后缀名为.js的文件中,然后使用script标签来引用.

提示:

  • Java(由Sun发明)是更复杂的编写语言.
  • ECMA-262是JavaScript标准的官方名称.
  • JavaScript由Brendan Eich发明.它于1995年出现在 Netscape中(该浏览器已停止更新),并于1997年被ECMA(一个标准协会)采纳.

块级元素

什么是块级元素？在html中< div>、 < p>、< h1>、< form>、< ul> 和 < li>就是块级元素.设置display:block就是将元素显示为块级元素.
如下代码就是将内联元素a转换为块状元素,从而使a元素具有块状元素特点.

块级元素特点：

  • 每个块级元素都从新的一行开始,并且其后的元素也另起一行.

  • 元素的高度、宽度、行高以及顶和底边距都可设置.

  • 元素宽度在不设置的情况下,是它本身父容器的100%（和父元素的宽度一致),除非设定一个宽度.

内联元素

在html中,< span>、< a>、< label>、 < strong> 和< em>就是典型的内联元素(行内元素)(inline)元素.当然块状元素也可以通过代码display:inline将元素设置为内联元素.
如下代码就是将块状元素div转换为内联元素，从而使div元素具有内联元素特点.

内联元素的特点:

  • 和其他元素都在一行上.
  • 元素的高度、宽度及顶部和底部边距不可设置.
  • 元素的宽度就是它包含的文字或图片的宽度,不可改变.

内联块状元素

内联块状元素(inline-block)就是同时具备内联元素、块状元素的特点,代码display:inline-block就是将元素设置为内联块状元素.(css2.1新增),< img>、< input>标签就是这种内联块状标签.

内联块状元素的特点：

  • 和其他元素都在一行上；
  • 元素的高度、宽度、行高以及顶和底边距可设置。

盒模型

在浏览器下,每一个html元素都会被解析为一个装有东西的盒子.盒子本身有自己的边框(border),盒子里的内容到盒子边框的距离称为填充(padding),盒子边框与其它盒子之间的距离为边界(margin).
在css模型设计中,元素真实的宽度和高度不仅仅由width和height来决定,还包括内边距、外边距、边框来组成.

边框

盒子模型的边框就是围绕着内容及补白的线,这条线你可以设置它的粗细和样式和颜色.

  • border-style(边框样式)常见样式有:dashed(虚线)dotted(点线)solid(实现).
  • border-color(边框颜色)中的颜色可设置为十六进制颜色.

  • border-width(边框宽度)中的宽度可以设置为:thin,medium,thick(但不是很常用),最常用的是像素(px).
