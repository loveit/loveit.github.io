JavaScript 数组对象

数组是连续存储多个数据的存储空间.现实中数据都是批量分类管理,然而只有批量管理多个数据就要用数组保存.
数组是存储一组”相同数据类型”的数据结构;
数组使用下标方式来获取某一项数值.

  • 使用第2种方法创建Array对象时,由于指定了数组的长度,因此在对数组赋值时,元素个数不能超过其指定的长度；
  • 使用第3种方法创建Array对象时,数组长度由数组元素的个数决定.

数组元素的赋值与获取

  • 数组元素的赋值有两种方法:

1.在创建Array对象时直接赋值;

2.利用Array对象的元素下标最数组进行赋值;

这个方法直接向Array对象中输入元素值,或者是修改数组中的任意元素值.

  • 数组元素的获取:获取某一项的值都是通过数组元素的下标来获取.
