认识DOM

文档对象模型 DOM (Document Object Model)定义访问和处理HTML文档的标准方法. DOM 将HTML文档呈现为带有元素、属性和文本树结构(节点数).

节点层次图:

  • HTML文档由节点构成的集合,DOM节点有:

1.元素节点:上图中< html>、< body>、< p>等都是元素节点,即标签.

2.文本节点:向用户展示的内容,如< li>…中的JavaScript、DOM、CSS等文本.

3.属性节点:元素属性,如< a>标签的链接属性.
