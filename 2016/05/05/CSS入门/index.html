CSS基础

CSS是什么?CSS全称为”层叠样式表(Cascading Style Sheets)”,主要定义HTML内容在浏览器内显示的样式,如文字大小,颜色,字体加粗等.

当然,我们已经知道HTML,CSS和JavaScript是前端技术最核心的三个元素.HTML控制网页的结构,CSS控制网页的外观,JavaScript控制网页的行为.

CSS选择器

在 CSS 中，选择器是一种模式，用于选择需要添加样式的元素。

在{}之前的部分是”选择器”,”选择器”指明了{}中的”样式”的作用对象,也就是”样式”作用于网页中的哪些元素.

标签选择器:

就是html代码中的标签.如:< html>,< body>,< h1>,< p>,< img>.

为p标签设置12px字号,行间距设置1.6em样式.

类选择器:

在css编码中最常用到的.

<span style="color:red";font-size:100>.</span>类选择器名称{css样式代码;}
**英文圆点开头**,class="类选择器名称".

ID选择器:

为标签设置id=”ID名称”,而不是class=”类名称”.ID选择符前面是井号(#)号,而不是英文圆点(.).
要注意的是ID选择器只能在文档中使用一次,类选择器可以使用多次.
