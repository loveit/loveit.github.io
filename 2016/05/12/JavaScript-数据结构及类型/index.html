JavaScript-数据结构及类型

发表于 2016-05-12   |   分类于 JavaScript   |     |   阅读次数

昨天了解了JavaScript的简介,今天该继续了解JavaScript的数据结构和数据类型.

数据结构

标识符

指变量、函数、属性的名字,或者函数的参数.

  • 第一个字符必须是一个字母、下划线(_)或一个美元符号($);
  • 变量名不能包含空格、加号、减号等符号;
  • 标识符不能喝JavaScript中用于其他目的的关键字同名.

关键字

指在JavaScript语言中有特定含义,成为JavaScript语法中一部分的字.

常量

指一旦创建,值不可改变的特殊变量.主要用于为程序提供固定和精确的值,包括数字和字符串,如数字、逻辑值真（true）、逻辑值假（false）等都是常量。

变量

是存储信息的容器.

  1. 变量的命名:实际上是一个标识符,因此命名一个变量时也要遵循标识符的命名规则.
  • 第一个字符必须是字母、下划线（_）或美元符号,其后的字符可以是字母、数字或下划线、美元符号;
    • 变量名不能包含空格、加号、减号等符号；
    • 标识符不能和JavaScript中用于其他目的的关键字同名；

  1. 变量的声明与赋值
    所有的JavaScript变量都由关键字var声明.一个var也可以同时声明多个变量名,变量名之间必须用英文逗号’,’隔开.

  2. 变量的作用域:指某变量在程序中的有效范围,也就是程序中定义这个变量的区域.变量区域可分为:全局变量和局部变量.

JavaScript数据类型

定义:变量中存储的数据的类型.是弱数据的方式.

JavaScript数据类型有两大类,一是基本数据类型,二是特殊数据类型.

基本数据类型:
number:数字型
string:字符串型
boolean:布尔型

特殊数据类型:
null:空值型
undefined:未定义
转义字符

Number 类型

number是最基本的数据类型.不区别整型数值(int)和浮点型数据(float).

  • 整型数据:是整数没有小数的,是数据形式十进制整数来的,整数可以为正数、0或负数.
  • 浮点型数据:带有小数点的数据,还可以使用指数法来表示,即实数后跟随字母e或E,后面加上正负号.

String 类型

用于表示由零个或多个字符组成的序列.字符串可以由双引号(“)或单引号(‘)表示.

Boolean 类型

仅有两个值得特殊数据类型:true、false.这两个值与数字值不是一回事,因此true不一定等于1,而false也不一定等于0.

null 类型

是只有一个值的数据类型.null值表示一个变量被赋予了一个空值.
null不等同于空的字符串(“”)或0,因为空的字符串(“”)或0是存在的,但是null表示其不存在的.

undefined 类型

undefined表示一个变量仅声明过,但从未被赋值.

转义字符

专门表示非打印字符以及特殊符号.

\b    退格;
\n    回车换行;
\t    Tab符号;
\f    换页;
\'    单引号;
\"    双引号;
\v    跳格（Tab，水平）;
\r    换行;
\\    反斜杠;
\OOO    八进制整数，范围为000~777;
\xHH    十六进制整数，范围为00~FF;
\uhhhh    十六进制编码的Unicode字符.
