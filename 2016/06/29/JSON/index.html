JSON 简介

JSON,全称”JavaScript Object Notation（JavaScript对象表示法)”,是一种数据格式.

对象结构

对象结构是使用大括号”{}”括起来的,大括号内是由0个或多个用英文逗号分隔的”关键字:值”对(key:value)构成的.

jsonObj指的是json对象.对象结构是以”{”开始,到”}”结束.其中”键名”和”值”之间用英文冒号构成对,两个”键名:值”之间用英文逗号分隔.键名是字符串值可以是数值、字符串、对象、数组或逻辑true和false.

数组结构

JSON数组结构是用中括号”[]”括起来,中括号内部由0个或多个以英文逗号”,”分隔的值列表组成.

语法:

说明:arr指的是json数组.数组结构是以“[”开始,到“]”结束,这一点跟JSON对象不同.不过在JSON数组结构中,每一对“{}”相当于一个JSON对象,这里的键名是字符串,值可以是数值、字符串、对象、数组或逻辑true和false.

对象和字符串

1.普通字符串: 单引号或双引号括起来的一串字符串.

2.JSON对象:指符合JSON格式要求的JavaScript对象.

3.JOSN字符串:指的是符合”JOSN格式”的字符串.

4.使用JOSN.stringify()方法将JOSN对象转换为JOSN字符串.

5.使用JSON.parse()的方法来将字符串转换为JOSN对象.
