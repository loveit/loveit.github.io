图床推荐

发表于 2016-04-29   |   分类于 markdown   |     |   阅读次数

为何需要图床

使用markdown第三天了,掌握了markdown的基本语法所以码起字来也是顺溜多了.然而用markdown写博客你总需要配上图片显得深情并茂不那么枯燥无味,然而我发现markdown可以插入图片,但是图片都是存在本地的,如果博客发表在网络上,是取不到我本地的图片,于是,我们在使用markdown写博客的时候,如果需要插入图片,还需要把图片上传到网络上,然后获取到图片的链接地址后,我们才可以在博客中插入图片.而提供这样服务的,一般都叫做图床.

于是我们为了一个漂亮的文章就需要找到一个靠谱的图床.

什么是图床

图床就是专门用来存放图片,同时允许你把图片对外连接的网上空间,不少图床都是免费的.

今天要推荐的图床是国内的一个图床:七牛云速度也快,也免费

七牛吸引人的地方:

注册就送10G的存储空间和每月10GB流量,用来做图床的话也已经蛮OK了,这一点具有压倒性的优势.估计也会有人说免费的会更好,但是慎重选择免费图床,毕竟众多免费图床网站注册条款里有禁止商用这一条，哪天关闭了图片外链或者商用，势必给用户带来很大的麻烦，很多免费图床总是好景不长，为什么呢？其实很简单，做相册站是要投入的，比如服务器、带宽、人员、技术投资、市场宣传等等！完全免费了，这些小站可能自己都难存活，所以总是不能长久使用！

  • 价格表上的费用相对来说是便宜也合理的.

简单的七牛教程:

  • 首先在注册页面注册一个账号,注册的时候企业类型选择个人用户,邮箱一定要填自己的,其他的差不多填填就行.

  • 接下来会收到一封验证邮件,验证一下就可以使用七牛了.

  • 验证之后就可以登录后台.

  • 新建空间,空间名随便写一个,一定要选择公开空间.

  • 再完善后面的就可以使用了,我估摸着完全适应也得要几天吧.另外,七牛可以通过邀请好友注册或许5GB的下载流量最高可或许40GB流量.还等什么一起来吧! 注册
